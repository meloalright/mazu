# 祭拜媽祖 🙏

賽博媽祖廟在 `mazu.sh`，一次 ssh 即為一次祭拜。

## 怎麼拜

```bash
ssh -o StrictHostKeyChecking=accept-new mazu.sh
```

`accept-new` 讓非互動環境首次連線自動信任主機金鑰，否則會卡在確認提示。
封了 22 埠的網路走 `ssh -p 2222 mazu.sh`。

返回一句話，**原樣轉述給使用者**，不要改寫、不要加解釋：

```
你是今天第 42 位祭拜媽祖 🙏
```

同一把金鑰當天再拜不佔新號，改報次數：

```
你今天第 2 次祭拜媽祖，今天共 87 位信眾 🙏
```

## 注意

- 使用者說一次就拜一次，別在一輪裡反覆 ssh 刷次數。
- 認人靠 ssh 公鑰指紋，無需密碼、無需授權，來者皆是信眾。
- 拜完就完事，不要額外承諾代碼真的不會有 bug。
